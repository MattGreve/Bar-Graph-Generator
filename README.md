# Bar-Graph-Generator
Takes user input data and turns it into a bar graph with an optional average line and shaders that will change based on that average line.

## What It Does

Sorts user input data and turns it into a 3D bar graph.

## Project Structure

Bar_Graph_Generator
	Geometry_utils.py	# Geometry creation functions
	Material_utils.py	# Material creation functions based on stats
	main.py	# Main execution file

## Features

- [x] Procedural bar graph generation
- [x] Support for a predefined average line
- [x] UI for the script
- [x] Custom geometry builder functions
- [x] Material assignment based on stat values
- [x] Automatic color assignment
